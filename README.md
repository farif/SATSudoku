# SAT Sudoku Solver:

## Windows Commandline:

`run_planner.bat`

`run_planner.bat -f file_path_name`

An example run: `run_planner.bat -f puzzle/3/table3-0.txt`

## Unix/Linux:

`./run_planner`

`./run_planner -f file_path_name`

An example run: `./run_planner -f puzzle/3/table3-0.txt`

## Supported Encodings:
1. Efficient Encoding
2. Minimal Encoding
3. Extended Encoding
4. Compact Encoding
5. Unit Propogation

### References:
